## String
### **toUpperCase(), toLowerCase()**
* toUpperCase() : 대문자로 바꾸기
```
String str = "string";
str = str.toUpperCase();  // STRING
```

* toLowerCase : 소문자로 바꾸기
```
String str = "STRING";
str = str.toLowerCase();  // string
```
